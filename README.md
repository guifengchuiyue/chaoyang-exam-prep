# 🎯 北京市朝阳区社区工作者全真模拟与专项刷题系统

> 基于 Streamlit + SQLite + 大模型 API 的全自动备考平台。
> 内置 5 大板块考纲逻辑，点击即自动生成高仿真题目，无需手动上传任何资料。

---

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置 API Key

在项目根目录创建 `.streamlit/secrets.toml`：

```toml
LLM_API_BASE_URL = "https://api.deepseek.com/v1"
LLM_API_KEY = "sk-你的密钥"
LLM_MODEL_NAME = "deepseek-chat"
```

或在应用启动后，在左侧边栏「⚙️ API 配置」中直接填入。

### 3. 启动

```bash
streamlit run app.py
```

浏览器打开 `http://localhost:8501`

## 📋 功能模块

### 📝 五大专项训练（Tabs 切换）

| 板块 | 考查内容 |
|------|---------|
| 🏛️ 公共管理 | 行政理论、基层自治、接诉即办、网格化管理 |
| ⚖️ 法律基础 | 《民法典》邻里/物业/继承、《居委会组织法》、《劳动法》 |
| 🤝 社会建设与社会工作 | 个案/小组/社区工作、社会保障、矛盾化解 |
| 🏙️ 朝阳区区情 | 区划地理、CBD、"两区"建设、"五宜"朝阳 |
| 🧠 基本能力 | 逻辑推理、言语理解、应急处理、群众沟通 |

每个板块点击即自动调用 AI 生成 5 道题目（3单选+1多选+1判断），无需手动输入资料。

### 📋 全真模拟考试

- **一键生成** 20 题综合模拟卷（5 大板块各 4 题）
- **⏳ 倒计时** 交卷
- **📊 雷达图 + 柱状图** 分析各板块得分
- **🔍 逐题解析** + 考点标注
- 自动保存成绩历史

### 📓 错题本

- 自动收录错题，按板块筛选
- **错题重做**模式：答对自动移除，答错继续保留
- 掌握率统计

### 📊 学习报告

- 考试成绩趋势图
- 每日刷题统计
- 错题板块分布饼图
